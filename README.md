
# Iniparser

## I - Overview

This modules offers parsing of ini files from the C++ level.

Key features :
- C++ interface
- Single class for parsing ini files

## II - Building project

A simple `make` at the root of the project should be enough to get the static
(i.e. `libinip.a`) library compiled.
